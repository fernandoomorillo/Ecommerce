# Ecommerce

## Objetivo
Desarollar una página web de Ecommerce, yo me he enfocado en una tienda de camisetas de fútbol retro de diferentes equipos y selecciones 

## Funcionalidades que tiene que tener el proyecto:
- Gestión del carrito: crear una función que reciba un id de un producto y lo guarde en un array
- Calculo del total con el objetivo de sumar precios para el cálculo del envío
- Filtrar por una categoría
- ORdenar los productos de menos a mayor precio
- Simulación de la sesión de usuario

## Tecnologías usadas
Para el desarrollo de la página web se ha utilizado:
- HTML
- CSS
- JS
